# Yazi Configuration

> Fast terminal file manager with Vim keybindings and Catppuccin theme

## Quick Reference

- **Config Path**: `~/.config/yazi/`
- **Documentation**: [docs/services/yazi.md](../docs/services/yazi.md)
- **Key Features**: Rust-based, 3-panel layout, file preview, transparent background
- **Navigation**: hjkl (Vim-style), `/` search, `Space` select

## Quick Start

```bash
# Install
cargo install --locked yazi-fm yazi-cli
# or: brew install yazi

# Apply config
cd ~/dotfiles && stow yazi
```

For complete documentation, see [docs/services/yazi.md](../docs/services/yazi.md)
