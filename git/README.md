# Git Configuration

> Git with Delta diff viewer and Catppuccin Mocha theme

## Quick Reference

- **Config File**: `~/.gitconfig`
- **Documentation**: [docs/services/git.md](../docs/services/git.md)
- **Key Features**: Delta for diffs, side-by-side view, syntax highlighting, nvimdiff merge tool
- **Theme**: Catppuccin Mocha with line numbers

## Quick Start

```bash
# Apply config
cd ~/dotfiles && stow git
```

For complete documentation, see [docs/services/git.md](../docs/services/git.md)
