# Docker Configuration

> Docker completion for Zsh shell

## Quick Reference

- **Config Path**: `~/.docker/completions/`
- **Documentation**: [docs/services/docker.md](../docs/services/docker.md)
- **Key Features**: Intelligent autocompletion, option-stacking, compose support

## Quick Start

```bash
# Apply config
cd ~/dotfiles && stow docker

# Reload shell
exec zsh
```

For complete documentation, see [docs/services/docker.md](../docs/services/docker.md)
