# Starship Configuration

> Minimal, fast prompt with Catppuccin Mocha theme and contextual information

## Quick Reference

- **Config File**: `~/.config/starship.toml`
- **Documentation**: [docs/services/starship.md](../docs/services/starship.md)
- **Key Features**: Rust-based, Git integration, Vi mode indicator, OS icon
- **Modules**: OS, hostname (SSH), directory, Git branch/status, Node.js, duration

## Quick Start

```bash
# Install
curl -sS https://starship.rs/install.sh | sh

# Apply config
cd ~/dotfiles && stow starship
```

For complete documentation, see [docs/services/starship.md](../docs/services/starship.md)
