# WezTerm Configuration

> GPU-accelerated terminal emulator with Catppuccin Mocha and development-focused features

## Quick Reference

- **Config File**: `~/.config/wezterm/wezterm.lua`
- **Documentation**: [docs/services/wezterm.md](../docs/services/wezterm.md)
- **Key Features**: GPU-accelerated, transparent background, Quick Select, hyperlinks
- **Font**: JetBrainsMono Nerd Font 14.5pt
- **Tab Bar**: Hidden (use tmux for multiplexing)

## Quick Start

```bash
# Install (macOS)
brew install --cask wezterm
brew install --cask font-jetbrains-mono-nerd-font

# Apply config
cd ~/dotfiles && stow wezterm
```

For complete documentation, see [docs/services/wezterm.md](../docs/services/wezterm.md)
