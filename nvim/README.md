# Neovim Configuration

> Modular Neovim with lazy.nvim, LSP, and Catppuccin Mocha theme

## Quick Reference

- **Config Path**: `~/.config/nvim/`
- **Documentation**: [docs/services/nvim.md](../docs/services/nvim.md)
- **Key Features**: Lazy.nvim plugin manager, Mason LSP, Telescope, LazyGit, nvim-cmp
- **Theme**: Catppuccin Mocha with transparency
- **AI**: Supermaven autocompletion (1M token context, free)

## Quick Start

```bash
# Apply config
cd ~/dotfiles && stow nvim

# Open Neovim (plugins auto-install)
nvim

# Manage plugins
# :Lazy sync - Install/update all
# :Mason - Install LSP servers
```

For complete documentation, see [docs/services/nvim.md](../docs/services/nvim.md)
