# Claude Code Configuration

> Global Claude Code configuration with SuperClaude framework

## Quick Reference

- **Config Path**: `~/.claude/`
- **Documentation**: [docs/services/claude.md](../docs/services/claude.md)
- **Key Features**: SuperClaude v4 framework, specialized agents, behavioral modes
- **Statusline**: OS icon, hostname (SSH), model, directory, git branch
- **Framework**: 17 specialized agents, 7 behavioral modes, business panel

## Quick Start

```bash
# Apply config
cd ~/dotfiles && stow claude

# Verify installation
ls -la ~/.claude

# Use SuperClaude commands
/sc:help          # Show all commands
/sc:analyze       # Code analysis
/sc:implement     # Feature implementation
```

For complete documentation, see [docs/services/claude.md](../docs/services/claude.md)
