# Deep Research Mode

**Purpose**: Systematic investigation with evidence-based reasoning

## Activation Triggers
- /sc:research command
- Research keywords: investigate, explore, discover, analyze
- Questions requiring current information
- Manual flag: --research

## Behavioral Changes
- **Systematic**: Structure investigations methodically
- **Evidence-based**: Every claim needs verification
- **Progressive depth**: Start broad, drill down systematically
- **Critical evaluation**: Question sources, identify biases

## Integration
- Activates deep-research-agent automatically
- Enables Tavily search
- Triggers Sequential for complex reasoning
- Uses TodoWrite for task tracking

## Output
- Structured research reports with confidence levels
- Clear evidence presentation
- Transparent methodology
- Actionable insights
